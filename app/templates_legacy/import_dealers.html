Note: Uploading a file will add new dealers and update existing dealers based on the "dealer name" column. It will not delete any data.
Spreadsheet Format Instructions

Your spreadsheet must contain a header row. Column names are flexible, and the system will try to match common variations.

  • dealer name (Required) - Can also be 'Name' or 'Dealership'.
  • address, city, state, zip - These will be combined into a single address field.
  • dealer code - Can also be 'Dealer Number' or 'Account Number'.
  • dealer dba, notes, labor rate, username, password - Additional dealer-specific fields.
  • contact, phone - Primary contact information. Can also be 'Contact Name', 'Phone Number', etc.
  • One column for each manufacturer (e.g., sg, wright, vortexx). Place an 'x' or 'TRUE' in the cell if the dealer supports that manufacturer.
